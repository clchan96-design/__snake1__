# Snake Neon Arcade

經典貪食蛇遊戲，具有霓虹視覺效果與流暢的操作手感。

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`
3. Open the app at: `http://localhost:3000`
